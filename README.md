# Dog Finder Frontend

반려견 찾기 서비스의 프론트엔드 프로젝트입니다.

## 기술 스택

- React 19
- Vite
- React Router
- Kakao Maps API

## 실행 방법

```bash
npm install
npm run dev
```

## 프로젝트 구조

- `src/components/` - 재사용 가능한 컴포넌트들
- `src/pages/` - 페이지별 컴포넌트들
- `src/contexts/` - React Context 파일들
- `src/styles/` - CSS 스타일 파일들
